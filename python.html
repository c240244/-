Lesson 1: Introduction to Python

Pythonとは

Python（パイソン）は、シンプルで分かりやすい文法を持つプログラミング言語です。 初心者でも学びやすく、世界中で幅広く利用されています。.

Pythonは、個人の学習目的から企業の大規模システムまで、さまざまな場面で使われています。

知っていましたか？

Python は、ヘビではなく、イギリスのコメディ グループ Monty Python にちなんで名付けられました。

Pythonの特徴

Pythonには次のような特徴があります。

  • 文法がシンプルで読みやすい
  • 少ないコード量でプログラムを書ける
  • 初心者でも理解しやすい
  • Windows、macOS、Linuxで動作する
  • 多くの便利なライブラリが用意されている

これらの理由から、Pythonは最初に学ぶプログラミング言語として非常に人気があります。

Pythonでできること

Pythonを使うと、次のようなことが可能です。

  • WebサイトやWebアプリケーションの開発
  • データ分析やグラフの作成
  • 人工知能（AI）や機械学習の開発
  • ファイル操作や作業の自動化
  • ゲームやツールの作成

はじめてのPythonプログラム

Pythonで最も基本的な操作は、画面に文字を表示することです。

# writing hello, カットリソニス!
print("Hello, カットリソニス!")
# Output: Hello, カットリソニス!

Lesson 2: Variables & Data Types

What are Variables?

Variables are used to store data values. In Python, you don't need to declare a variable with a specific type. The type is inferred from the value you assign.

# Variable assignment
name = "John"
age = 25
height = 5.9
is_student = True

Basic Data Types

  • Integers: Whole numbers (e.g., 5, -3, 1000)
  • Floats: Decimal numbers (e.g., 3.14, -0.5, 2.0)
  • Strings: Text data (e.g., "hello", 'Python')
  • Booleans: True or False values
  • None: Represents absence of value
